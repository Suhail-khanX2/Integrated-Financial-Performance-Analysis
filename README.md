Financial Performance Analysis of Flex Ltd. (2020–21 to 2024–25)
This project analyzes the five-year financial performance of Flex Ltd. using its Profit and Loss Account and Balance Sheet. The goal is to understand how the company’s profitability, liquidity, solvency, efficiency, and overall financial health changed over time. The project also focuses on building clean, structured Excel models similar to real finance trainee work.
________________________________________
1. Project Overview
The project evaluates Flex Ltd.’s financial performance over five financial years. It includes preparation of ratio analysis, trend analysis, common-size statements, comparative statements, and key insights. The analysis highlights major patterns in revenue, expenses, profits, and working-capital movements.
